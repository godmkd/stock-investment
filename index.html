📊 投資總覽

💡 總覽使用說明
• 這裡顯示所有帳戶的投資摘要，包含美股、台股、房產的總資產和淨資產
• 使用上方「持有人篩選」可以查看個別持有人或全部合計
• 選擇「按戶名分開」可以看到每位持有人各自的圓餅圖和趨勢圖
• 圓餅圖顯示持股配置比例，趨勢圖可切換 30天/6個月/1年/自訂時間範圍
• 所有金額單位：美股為 USD，台股為 TWD，總覽中的合計以 TWD 計算

🇺🇸 美股投資組合

💡 美股投資使用說明
• 需要先到「帳戶管理」建立帳戶，才能記錄交易
• 輸入 Finnhub API Key 後可即時更新美股股價（免費申請，下方有連結）
• 「資金管理」記錄存入/提出美元資金，可填匯率自動換算台幣
• 「新增交易」記錄買賣操作，支援一次新增多筆
• 「股利 & 利息」記錄配息、利息、稅額和微調
• 持倉明細顯示現價、均價、損益，系統自動計算投報率
• 資料存在瀏覽器 localStorage，記得定期到「帳戶管理」做備份
!免費的 Finnhub API Key 可從 finnhub.io 申請。沒有 API Key 也能使用，只是無法自動更新美股股價。
免費申請 Finnhub API Key：https://finnhub.io/register（註冊後到 Dashboard 複製 API Key）

🇹🇼 台股投資組合

💡 台股投資使用說明
• 台股股價會自動更新（使用 TWSE 即時報價），也可按「🔄 更新台股股價」手動刷新
• 股票代號支援數字代碼（如 2330）或中文名稱（如 台積電）
• 交易金額以「整股」計算（股數 × 股價，無條件捨去到整數）
• 其餘操作方式與美股相同：先建帳戶 → 存入資金 → 新增交易

📅 定期定額快速輸入

💡 定期定額使用說明
• 快速輸入定期定額買入紀錄，每筆可設不同股票
• 股票代號自動判別市場：純數字 → 台股，含英文 → 美股
• 手續費率會依市場自動帶入（台股 0.1425%，美股 0.01425%）
• 每個帳戶可設不同金額，勾選要參與的帳戶
• 送出後自動寫入對應市場的交易紀錄

🏠 房產 / 房貸管理

💡 房產/房貸使用說明
• 支援「預售屋」和「成屋」兩種類型
• 預售屋有工程款分段付款功能，最後兩筆（倒數第二筆和交屋款）會自動聯動計算
• 工程款佔總價 % 和工程款總額可擇一填入，另一個自動帶入
• 貸款部分：填剩餘貸款或已繳本金，另一個自動計算
• 未手動填寫剩餘貸款時，系統自動帶入「總價 × 貸款比例」
• 可綁定帳戶（戶名），在總覽中會歸到該持有人名下

⚙️ 帳戶管理

💡 帳戶管理使用說明
• 帳戶名稱就是「持有人」，在總覽中會依此分類
• 每個帳戶可同時有美股和台股的交易紀錄
• 「完整備份」會把所有資料（帳戶、交易、房產、股價快取）打包下載為 JSON
• 「還原備份」可匯入之前備份的 JSON 檔，換電腦或清瀏覽器後可恢復資料
• ⚠️ 所有資料存在瀏覽器 localStorage，清除瀏覽器資料會遺失！建議定期備份
